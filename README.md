🗳️ Online Voting System (C++)

📌 Overview

This project is a console-based Online Voting System developed using C++. It allows users to cast votes securely while preventing duplicate voting using unique Aadhaar IDs. The system uses file handling to simulate a database and ensures data persistence.


🚀 Features

- 🗳️ Vote casting system
- 🔒 Duplicate vote prevention using Aadhaar ID
- 💾 File-based database (votes.txt, voters.txt)
- 📊 Real-time vote counting
- 🏆 Winner declaration with tie handling
- 📋 Formatted result display using setw()


🛠️ Technologies Used

- C++
- STL (map, unordered_set)
- File Handling (fstream)
